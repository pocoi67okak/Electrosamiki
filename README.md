# Electrosamiki

Spigot-плагин для Minecraft 1.21.8, который добавляет электросамокаты.

## Версия 2.0.1

- Исправлено движение: самокат теперь едет через `setVelocity()` каждый тик, а не через teleport сущности с пассажиром.

## Версия 2.0.0

- Самокат теперь не обычная вагонетка: основа сделана на невидимом armor stand.
- Добавлена ванильная модель без ресурс-пака через block display entities.
- Езда стала ровнее: движение идет телепортом каждый тик по направлению взгляда игрока, без физики minecart без рельс.
- Старые minecart-самокаты из версии 1 автоматически мигрируют в новый формат при загрузке чанка.

## Возможности

- `/electrosamik give <ник>` выдает игроку предмет электросамоката.
- Правый клик предметом по блоку ставит самокат.
- Правый клик по самокату сажает игрока.
- Во время езды клик любой кнопкой мыши переключает скорость:
  `3 -> 4 -> 5 -> 6 -> 7 -> 8 -> 3` блоков в секунду.
- Если сломать самокат, выпадает такой же предмет электросамоката.

## Команды

| Команда | Описание |
| --- | --- |
| `/electrosamik give <ник>` | Выдать электросамокат игроку |

## Права

| Право | По умолчанию | Описание |
| --- | --- | --- |
| `electrosamiki.command` | все | Доступ к базовой команде |
| `electrosamiki.give` | операторы | Выдача электросамокатов |
| `electrosamiki.use` | все | Установка, езда и снятие самокатов |

## Установка

1. Откройте GitHub Actions в репозитории.
2. Скачайте артефакт `Electrosamiki` из последней успешной сборки.
3. Поместите jar-файл в папку `plugins` сервера.
4. Перезапустите сервер.

## Сборка

Jar собирается в GitHub Actions на Java 21. Workflow находится в `.github/workflows/build.yml`.

Локально плагин собирать не требуется.

## Требования

- Minecraft 1.21.8
- Spigot/CraftBukkit API 1.21.8-R0.1-SNAPSHOT
- Java 21
